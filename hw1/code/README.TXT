When you run singleyLinkedList, doublyLinkedList, and circularLinkedList, you will be presented with a list of functions that can be performed. Enter the number associated with the function you would like to perform followed by a space and the integer you wish for the function to utilize. All the linked Lists start empty.

When you run binarySearh Tree you will be presented with a list of functions that can be performed. Enter the number associated with the function you would like to perform followed by a space and the integer you wish for the function to utilize. The Binary Search Tree Starts with 8 nodes already in it. 

When you run the prioirtyQueue there is no interface, a series of commands will run and execute showcasing the four functions that are implemented.
